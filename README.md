# 네온 테트리스 — VS (PWA)

솔로 + 2인 대전 테트리스. 방 코드로 친구와 한판. 아이폰·갤럭시 홈화면에 앱처럼 설치돼요.

## 파일 구성

```
tetris-app/
├─ index.html              ← 게임 본체
├─ manifest.webmanifest    ← 앱 이름/아이콘/색상 정의
├─ sw.js                   ← 서비스워커 (설치·오프라인 캐시)
├─ mqtt.min.js             ← VS 통신용 라이브러리 (로컬 호스팅)
├─ icon-180.png            ← 아이폰 홈화면 아이콘
├─ icon-192.png / 512.png  ← 안드로이드 아이콘
└─ icon-512-maskable.png   ← 안드로이드 적응형 아이콘
```

> **중요**: PWA "홈화면 설치"는 파일을 그냥 여는 걸로는 안 되고 **HTTPS 주소로 띄워야** 됩니다. 아래처럼 한 번만 올리면 끝.

## 배포 (셋 중 하나)

**1) Vercel — 제일 쉬움**
- [vercel.com](https://vercel.com) 로그인 → `Add New → Project`
- `tetris-app` 폴더를 통째로 드래그앤드롭 (또는 이 폴더를 git에 올려 연결)
- 나오는 `https://....vercel.app` 주소가 게임 주소

**2) 기존 Vercel 프로젝트에 얹기**
- 쓰던 프로젝트 폴더에 `tetris-app/` 통째로 복사 → `https://기존주소/tetris-app/` 로 접속

**3) GitHub Pages**
- 새 repo에 파일들 push → `Settings → Pages → Branch: main` → 발급된 주소 사용

## 홈화면 설치

**아이폰 (Safari)**
1. Safari로 게임 주소 열기 (⚠️ 크롬 말고 Safari)
2. 공유 버튼 → `홈 화면에 추가`
3. 홈화면 아이콘으로 실행하면 전체화면 앱처럼 동작

**갤럭시 (Chrome)**
1. Chrome으로 게임 주소 열기
2. 자동으로 뜨는 `앱 설치` 배너 탭, 또는 우상단 ⋮ → `앱 설치`
3. 앱 서랍/홈화면에 추가됨

## 둘이서 플레이

1. 두 폰 모두 앱 실행 → `VS 모드`
2. 한 명: `방 만들기` → 4자리 코드 나옴
3. 다른 한 명: `방 참여` → 그 코드 입력
4. 연결되면 3·2·1 카운트다운 후 대전 시작
5. 줄을 지우면 상대에게 쓰레기 줄을 보냄. 먼저 죽으면 패배.

## 알아둘 점

- **VS 모드는 인터넷이 필요해요.** 통신은 공용 MQTT 브로커(`broker.emqx.io`)를 거칩니다. 둘 다 와이파이/LTE만 되면 OK (같은 네트워크일 필요 없음).
- **솔로 모드는 오프라인에서도** 됩니다 (서비스워커가 캐시). 한 번 접속해두면 비행기모드에서도 실행 가능.
- 공용 브로커라 아주 드물게 방 코드가 겹칠 수 있어요. 안 붙으면 방을 새로 만들면 됩니다.
- 게임 로직·디자인·조작은 기존 파일 그대로 유지했고, PWA 설치 기능만 입혔습니다.
  - 바뀐 것: mqtt를 CDN 대신 로컬 호스팅(로딩 안정), manifest·아이콘·서비스워커 추가, iOS/안드로이드 설치 메타태그 추가.
